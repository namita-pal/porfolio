About Project

Developed an e-commerce cosmetic website using WordPress, PHP, and MySQL. The platform enables users to browse products, add items to the cart, and complete secure checkouts. It includes dedicated sections for men and women, improving navigation, user experience, and boosting online sales through well-organized product categorization and seamless functionality.

Key Features

  • User-Friendly Interface: Attractive and responsive design built with WordPress for smooth navigation.
  • Product Browsing:Categorized listings for easy product discovery.
  • Add-to-Cart System: Allows users to add, update, or remove products before checkout.
  • Gender-Based Sections: Separate sections for men’s and women’s cosmetics to enhance shopping experience.
  • Database Integration: MySQL ensures secure storage of user and order data.

Technologies Used

  • WordPress
  • PHP
  • MySQL
